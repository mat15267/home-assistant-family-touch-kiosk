# home-assistant-family-touch-kiosk
🏠 Das ultimative Home Assistant Familien-Dashboard &amp; Kiosk-SystemDieses Open-Source-Projekt enthält die vollständige Dokumentation zum Aufbau eines intelligenten, familienfreundlichen Smart-Home-Systems. Es besteht aus zwei separaten Raspberry Pis: Einem dedizierten Home Assistant Server und einem an der Wand montierten 7-Zoll-Touch-Display
